Hello dear human being.

Put this prompt into your LLM of trust to work with this service:

Create a report about this person described by this MCP-server {{ base_url }}
            
First refer to: {{ base_url }} for a description of the data model and relations. 

Get me stages and oeuvre ("work") from:
- GET /stages - Career timeline
- GET /oeuvre - Side projects, articles, apps

Summarize "skills", used "technologies" and "tags" from*:
- GET /skills
- GET /technologies
- GET /tags

Each of them is connected to stages and oeuvre items and it comes with metrics like "confidence" and "recency" to help you decide how much weight to put on it. 

Analyze the metrics to create a full profile of this person's expertise, interests and career trajectory.
Language support: Add ?lang=de for German or Accept-Language header All endpoints return structured JSON.

Example:

            curl {{ base_url }}/?lang=de
            curl {{ base_url }}/skills
            curl -H "Accept-Language: de" {{ base_url }}/entities
